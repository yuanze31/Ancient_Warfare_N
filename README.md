# 如何使用本仓库

1. 在```main/awB/awScripts```下，自行补全```QuModLibs```库  
   如果操作正确，目录应为```main/awB/awScripts/QuModLibs/QuMod.py```
2. 正常导入```MCStudio```，所选目录应为```main/```，其中```main/awB```为行为包，其中```main/awR```为材质包
